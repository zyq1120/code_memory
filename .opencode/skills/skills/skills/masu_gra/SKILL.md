# 马鞍山学院毕业论文格式规范

## 一、封面格式

### 1. 学校名称
- 第一行：马鞍山学院 Ma´anshan University
- 字体：仿宋，小二，居中

### 2. 届数
- 内容：2026届本科毕业设计（论文）
- 字体：仿宋，小二号，居中

### 3. 题目
- 论文题目不得超过20个字
- 要简练、准确，可分为两行
- 字体：黑体，二号，居中

### 4. 学生信息栏
包含以下信息项，专业班级必须使用全称，不允许简写：
- 学 生 姓 名
- 学 号
- 专 业 班 级（全称，如"软件工程软升243"）
- 学 院
- 指 导 教 师
- 完 成 日 期

### 5. 页脚
- 教务处 制

---

## 二、声明页格式

### 1. 原创性声明
- 标题：马鞍山学院学位论文原创性声明（加粗居中）
- 正文：仿宋体，声明内容
- 签名栏：学位论文作者 + 日期（格式：2026年4月20日）

### 2. 授权书
- 标题：马鞍山学院学位论文版权使用授权书（加粗居中）
- 保密选项：勾选框格式

---

## 三、任务书格式

### 标题
- 毕业设计(论文)任务书（加粗居中）

### 主要内容及要求
- 主要内容：编号列表 (1)-(5)
- 论文要求：编号列表 (1)-(3)

### 签字栏
- 指导教师签字 + 日期

---

## 四、摘要格式

### 1. 标题格式
- 中文："摘 要"（两字之间空4格）
- 英文："Abstract"

### 2. 字数要求
- 摘要字数控制在300-500字之间

### 3. 内容结构
- 研究背景与问题
- 研究方法与技术路线
- 主要成果与结论

### 4. 关键词
- 格式：关键词：关键词1；关键词2；关键词3
- 数量：3-5个
- 英文：Key words: keyword1; keyword2; keyword3

---

## 五、目录格式

### 1. 层级结构
- 一级：摘要、Abstract、第N章 标题
- 二级：N.M 标题
- 三级：N.M.K 标题

### 2. 格式示例
```
摘 要 I
Abstract II
目 录 III
1 绪论 1
1.1 研究背景 1
1.1.1 国外研究现状 2
```

## 六、正文章节格式

### 1. 章节编号规则
```
第1章 绪论          （一级标题/章目）
1.1 研究背景        （二级标题）
1.1.1 国外研究现状  （三级标题）
```

### 2. 一级标题（章目）格式
- 字体：三号宋体加粗
- 题序和标题之间：空一格
- 行距：固定值20磅
- 段前：18磅
- 段后：18磅
- 对齐：居中

**示例**：
```
第1章 绪论
```

### 3. 二级标题格式
- 字体：四号宋体加粗
- 行距：固定值20磅
- 段前：12磅
- 段后：12磅
- 对齐：两端对齐，居左

**示例**：
```
1.1 研究背景
```

### 4. 三级标题格式
- 字体：小四号宋体加粗
- 行距：固定值20磅
- 段前：6磅
- 段后：6磅
- 对齐：两端对齐，居左

**示例**：
```
1.1.1 国外研究现状
```

### 5. 正文格式
- 字体：宋体，小四号
- 英文和数字：Times New Roman，小四号
- 行距：固定值18磅
- 段前：0磅
- 段后：0磅
- 对齐：两端对齐
- 首行缩进：2字符

**重要提示**：涉及到的数值和字母等都采用Times New Roman字体

## 七、图表格式规范

### 重要规则
- 图表编号统一使用 X.X 格式（如 图2.1、表5.1）
- 禁止使用 X-X 格式
- 图序要逐章单独编序，在章内必须连续，不得重复或跳跃

### 1. 图的格式

#### 基本要求
- 编号：图 X.X（如 图2.1）
- 标题位置：图下方
- 字体：宋体五号（数值和字母用Times New Roman）
- 行距：固定值18磅
- 对齐：居中

#### 图序与图标题
- 图序与图标题之间：空1格
- "图+图序+图标题"为一个整体

#### 与上下内容的间距
- 图与上文：空一行
- 图与下文：空一行

#### 分图标注
- 由若干个分图组成的插图，分图用 a, b, c, …… 标出

#### 图内字体
- 图内的字体不应过大，注意美观

**示例**：
```
[上文内容]

图2.1 系统架构图

[下文内容]
```

### 2. 表的格式

#### 基本要求
- 编号：表 X.X（如 表5.1）
- 标题位置：表上方
- 字体：宋体五号（数值和字母用Times New Roman）
- 行距：固定值18磅
- 对齐：居中

#### 表序与表标题
- 表序与表标题之间：空1格
- "表+表序+表标题"为一个整体
- 表序后空一格书写表题

#### 与上下内容的间距
- 表与上文：空一行
- 表与下文：空一行

#### 表序连续性
- 表序要逐章单独编序
- 表序在章内必须连续，不得重复或跳跃

#### 跨页表格处理
- 表格允许下页接写
- 表题可省略
- 表头应重复写
- 并在右上方写"续表××"

**示例**：
```
[上文内容]

表5.1 用户表结构设计

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | bigint | 主键 |
| name | varchar | 姓名 |

[下文内容]
```

## 八、参考文献格式

### 1. 数量要求
- 总数：12篇以上（近5年）
- 外文文献：不少于2篇
- 近三年参考文献：不少于6篇

### 2. 引用标注
- 正文引用：上标格式 [1-3, 5]
- 多个连续引用：[1-3]
- 多个不连续引用：[1, 3, 5]

### 3. 文献格式示例

#### 期刊论文
[序号] 作者. 文章标题[J]. 期刊名, 年份, 卷(期): 页码.

#### 书籍
[序号] 作者. 书名[M]. 出版地: 出版社, 年份: 页码.

#### 学位论文
[序号] 作者. 论文题目[D]. 学校, 年份.

#### 网络资源
[序号] 作者. 标题[EB/OL]. 网址, 访问日期.

### 4. 参考文献格式
- 字体：宋体五号
- 数值和字母：Times New Roman

## 十一、常见格式错误提醒

### 禁止项
1. ✗ 图表编号使用 X-X 格式（正确：X.X）
2. ✗ 专业班级使用简写
3. ✗ 论文题目超过20字
4. ✗ 摘要字数超出300-500字范围
5. ✗ 参考文献少于12篇或外文少于2篇
6. ✗ 近三年参考文献少于6篇
7. ✗ 正文使用口语化表达
8. ✗ 数值和字母使用宋体（应使用Times New Roman）
9. ✗ 图表与上下文未空行

### 必须项
1. ✓ 每章末尾有"本章小结"
2. ✓ 图表编号使用点号分隔（图2.1、表5.1）
3. ✓ 数字和英文使用Times New Roman
4. ✓ 摘要关键词使用分号分隔
5. ✓ 页码从正文开始
6. ✓ 一级标题题序和标题空一格
7. ✓ 图序/表序与标题空一格
8. ✓ 参考文献近5年，近3年不少于6篇
